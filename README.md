<h2> Instructions: </h2>

Run the program with:
```
poetry run invoke start
```

Run the tests with:
```
poetry run invoke test
```
Create the coverage report with:
```
poetry run invoke coverage-report
```
Run pylint tests with:
```
poetry run invoke lint
```
